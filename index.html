// system architecture

Autonomous agents, every technique

We treat autonomous discovery as a methods-agnostic problem, combining neuro-symbolic reasoning, evolutionary search, and interactive theorem proving rather than committing to a single paradigm.

01

Agentic planning & search

Agents decompose goals through hierarchical planning and explore the state space with Monte Carlo Tree Search, scaling solution quality with inference-time compute under an LLM-guided value function.

02

Recursive meta-optimization

A recursive self-improvement loop in which the agent meta-optimizes its own policies and search heuristics — refining the operators that generate subsequent candidates, not only the candidates themselves.

03

Verified synthesis

Candidates are constrained by a syntax-guided synthesis grammar and discharged against a semantic specification via static analysis and Lean 4 machine-checked proofs, excluding semantically invalid solutions.

// optimization engine

One recursive loop

An elite archive maps candidate agents to niches indexed by behavioral descriptors. A Task assigns each candidate a fitness; a Generator samples archived elites and emits mutated phenotypes; the illumination loop retains the per-niche elite — while the Generator's own variation operators remain subject to optimization.

GENERATOR ▸ CANDIDATE ▸ EVALUATE ▸ SCORE ▸ ARCHIVE

↑ the variation operator is itself meta-optimized — recursive adaptation of the search distribution

// empirical result

Self-adaptive σ outperforms a static scale

On a stochastic black-box landscape — only scalar fitness is observed, never the objective's structure — a generator performing online self-adaptation of its mutation scale (σ) outperforms a fixed exploration scale, consistent with cumulative step-size adaptation in CMA-ES.
